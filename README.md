# will j. jedrzejczak portfolio

interactive terminal portfolio website. click around to explore experience, projects, certifications, and more.

## what is this

a terminal-style portfolio where you can navigate through directories and files just like a linux terminal. no typing required, just click to explore.

## how to use

- click on directories (blue text with /) to navigate
- click on files (white text) to view content
- click ../ to go back to parent directory
- use ctrl+backspace to navigate up

## tech stack

- vanilla javascript
- html/css
- github pages
